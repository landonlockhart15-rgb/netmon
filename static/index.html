Router Setup — Point Your Network Here

For the ad blocker to work network-wide, your router needs to tell every device to use this PC as its DNS server. Each router is different — look for DHCP Settings or LAN Settings and set the Primary DNS to:

Detecting…
1Log in to your router (usually 192.168.1.1 or 192.168.0.1)
2Find LAN / DHCP Settings
3Set Primary DNS Server to the IP above
4Set Secondary DNS to 8.8.8.8 (fallback if this PC is off)
5Save and reconnect your devices (or wait for DHCP lease renewal)
What gets blocked: Ad network domains, tracking pixels, telemetry servers, known malware C&C domains.
What does NOT get blocked: YouTube/Netflix in-stream ads (those are served from the same domains as the content itself — no DNS blocker can block them without breaking the service).
This PC must stay on for DNS to work. The secondary DNS ensures devices still reach the internet if this PC is off.
