## Privacy checklist before committing images

Before adding a screenshot, check for:

- public IP addresses
- private IP addresses you do not want published
- MAC addresses
- device names
- Wi-Fi/network names
- usernames
- API keys or tokens
- notification URLs or credentials
- file paths that expose personal information

Blur or crop sensitive details before committing.
